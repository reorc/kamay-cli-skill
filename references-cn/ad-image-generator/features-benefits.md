# Features-Benefits 广告 Prompt 架构设计心法

> 核心理念：**功能即英雄，技术即故事**

## 一、策略 定位

Features-Benefits 类型广告的核心目标是**将产品功能转化为可视化的价值承诺**。不是简单罗列参数，而是让功能"活"起来，让用户在视觉上"看到"产品能力。

**关键心智模型**：
- 功能 → 视觉隐喻
- 参数 → 情感承诺
- 技术 → 信任符号

## 三、章节设计套路

### 3.1 UNEXPECTED ELEMENT（意外元素）

**设计原则**：打破品类视觉惯例，创造"我没见过这种广告"的瞬间

**常用套路**：

1. **反重力/倒置法**
   ```
   产品倒置悬浮，液体违反物理定律保持稳定
   → 隐喻：产品可靠到"违反常识"
   ```

2. **动态捕捉法**
   ```
   产品处于倾斜45度角，液体/蒸汽定格在运动瞬间
   → 隐喻：产品正在"工作"，不是静态展示
   ```

3. **功能可视化法**
   ```
   将抽象功能转化为视觉元素（如温度计融入文字、保温层发光）
   → 隐喻：技术看得见
   ```

4. **时间分割法**
   ```
   画面分割展示冷/热、早/晚、前/后对比
   → 隐喻：24小时恒温承诺
   ```

**示例模板**：
```
UNEXPECTED ELEMENT: Show the [product] positioned at a dramatic
[angle] as if caught mid-[action], with visible [effect] creating
a visual metaphor for [core benefit]. This creates visual surprise
and intrigue - viewers haven't seen [category] ads like this before.
```

### 3.2 COMPOSITION & EMOTIONAL STORY（构图与情感叙事）

**设计原则**：用视觉流引导眼球，用空间讲故事

**常用套路**：

1. **对角线构图**
   ```
   产品沿45度对角线放置，打破传统居中/左右分割
   → 创造动感和能量
   ```

2. **温度分割构图**
   ```
   上半部分冷色调（冰霜、蓝色），下半部分暖色调（金色、蒸汽）
   产品位于中心，成为"桥梁"
   → 视觉化温度控制能力
   ```

3. **视觉流设计**
   ```
   眼球路径：问题场景 → 产品 → 解决方案
   或：环境 → 产品 → 功能证明
   ```

**示例模板**：
```
COMPOSITION & EMOTIONAL STORY:
- Background: [Split/Gradient/Scene] composition showing [contrast]
- Product positioned at [location] as the [anchor/bridge/hero]
- Eye flow: [Start point] → [Product] → [End point/benefit]
- Creates visual tension that the product resolves
```

### 3.3 HEADLINE TREATMENT（标题处理）

**设计原则**：标题不是"贴"在画面上，而是"长"在构图里

**常用套路**：

1. **角度融合法**
   ```
   标题倾斜角度与产品倾斜角度一致
   → 文字成为视觉设计的一部分
   ```

2. **产品融合法**
   ```
   产品剪影替代文字中的某个字符（如"24"中的数字）
   → 产品即信息
   ```

3. **边缘定位法**
   ```
   标题沿画面边缘垂直排列，而非水平居中
   → 打破常规，创造记忆点
   ```

**核心参数强调**：
- 数字要**超大**（24小时、2度）
- 使用品牌色高亮关键词
- 中文简体，Hanken Grotesk 字体

**示例模板**：
```
HEADLINE TREATMENT:
- Main headline: "[核心承诺]" positioned [diagonally/vertically/integrated]
- Typography: Hanken Grotesk, bold, [angle] degrees rotation
- Key numbers ([24/2]) rendered EXTRA LARGE in brand color #a36710
- Text integrated INTO composition, not just overlaid
```

### 3.4 PRODUCT PRESENTATION（产品展示）

**设计原则**：产品必须"活着"——正在工作、正在解决问题

**常用套路**：

1. **动态状态展示**
   ```
   产品倾斜、悬浮、倒置、正在倾倒
   → 不是静态产品照，是"使用瞬间"
   ```

2. **功能可见化**
   ```
   透明展示内部结构（滤网、保温层）
   蒸汽/冷凝效果可见
   → 技术不是说的，是看到的
   ```

3. **人手元素**
   ```
   画面中出现手部，自然握持产品
   → 增加真实感和尺度感
   ```

**示例模板**：
```
PRODUCT PRESENTATION:
- Show the [product] in [dynamic state], NOT static
- Include visible evidence of function: [steam/condensation/glow]
- [Optional] Human hand element for scale and relatability
- Product appears ALIVE - actively performing its function
```

### 3.5 VISUAL DISTINCTIVENESS（视觉独特性）

**设计原则**：定义"明天还能记住"的那个元素

**常用套路**：

1. **单一记忆锚点**
   ```
   选择一个最强视觉元素作为记忆点
   （倒置产品、温度计文字、分割构图）
   ```

2. **色彩对比策略**
   ```
   冷暖对比、明暗对比、品牌色点缀
   → 在信息流中"跳出来"
   ```

3. **品类颠覆声明**
   ```
   明确说明"这与传统[品类]广告不同"
   → 给AI明确的创新方向
   ```

**示例模板**：
```
VISUAL DISTINCTIVENESS:
- The [specific element] is the memorable hook that stops scrolling
- Color contrast: [warm/cool] creates visual tension
- This breaks from typical [category] ads by [specific difference]
- Viewers will remember [specific visual] tomorrow
```

## 四、功能卖点视觉化套路

| 功能卖点 | 视觉化方式 |

## 六、避免的常见错误

1. **静态产品照** → 必须展示产品"在工作"
2. **功能文字堆砌** → 用视觉隐喻替代文字说明
3. **标题悬浮覆盖** → 标题必须融入构图
4. **居中对称构图** → 使用对角线/不对称创造动感
5. **忽略移动端** → 确保大字体、清晰层级
