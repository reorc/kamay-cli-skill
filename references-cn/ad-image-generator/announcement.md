# Announcement 广告 Prompt 架构设计心法

> 核心理念：**制造事件感，让产品成为"新闻"**

## 一、策略 定位

Announcement 类型广告的核心目标是**创造"值得关注"的时刻**，让用户感觉这是一个重要的产品发布或创新突破，而非普通的产品展示。

**关键心智模型**：
- 产品 → 创新事件
- 功能 → 突破性发现
- 广告 → 重要公告

**与其他 策略 的区别**：
| 维度 | Features-Benefits | Lifestyle | Announcement |
|------|-------------------|-----------|--------------|
| 情感 | 信任 | 向往 | **惊喜/期待** |
| 语气 | 说明 | 叙事 | **宣告** |
| 时态 | 现在 | 日常 | **此刻/全新** |
| 能量 | 稳定 | 温和 | **高能量** |

## 三、章节设计套路

### 3.1 UNEXPECTED ELEMENT（意外元素）

**设计原则**：创造"停止滚动"的视觉冲击

**Announcement 特有套路**：

1. **产品融入文字法**
   ```
   产品剪影成为标题的一部分
   如：产品替代"24小时"中的数字
   → 产品即信息，信息即产品
   ```

2. **时间冻结法**
   ```
   产品"冻结在时间中"
   展示温度/时间的可视化（冰晶、热浪定格）
   → 强调技术突破的瞬间
   ```

3. **对比冲突法**
   ```
   画面分割：冷/热、旧/新、问题/解决
   产品位于分割线，成为"解决方案"
   → 戏剧性的视觉张力
   ```

4. **动态悬浮法**
   ```
   产品悬浮、倾斜、处于运动状态
   打破静态产品照的常规
   → 能量感和创新感
   ```

**示例模板**：
```
UNEXPECTED ELEMENT:
- Create a [technique] that immediately captures attention
- The product [specific visual treatment]
- This visual surprise communicates [core message] without words
- Viewers stop scrolling because they've never seen [category] ads like this
```

### 3.2 COMPOSITION & EMOTIONAL STORY（构图与情感叙事）

**设计原则**：构建"发布时刻"的戏剧感

**常用套路**：

1. **聚光灯构图**
   ```
   产品位于视觉中心，周围元素向其汇聚
   光线、线条、视觉流都指向产品
   → 产品是"主角登场"
   ```

2. **对比分割构图**
   ```
   左右或上下分割，展示对比
   产品位于分割线，成为"桥梁"或"解决方案"
   → 戏剧性的问题-解决叙事
   ```

3. **能量爆发构图**
   ```
   从产品向外辐射的视觉元素
   （光线、粒子、能量波）
   → 产品是能量/创新的源头
   ```

4. **时间线构图**
   ```
   展示"之前"到"之后"的变化
   产品是这个变化的关键
   → 突破性创新的叙事
   ```

**示例模板**：
```
COMPOSITION & EMOTIONAL STORY:
- Structure: [Spotlight/Split/Radial/Timeline]
- Product position: [center/dividing line/source point]
- Visual flow: All elements [converge to/radiate from] product
- Emotional peak: [specific moment of impact]
- Creates sense of [discovery/breakthrough/announcement]
```

### 3.3 HEADLINE TREATMENT（标题处理）

**设计原则**：标题本身就是"公告"，要有宣言感

**常用套路**：

1. **超大数字强调**
   ```
   核心数字（24、2）超大展示
   成为视觉主体的一部分
   → 数据即承诺
   ```

2. **宣言式文案**
   ```
   使用肯定、有力的语气
   "终于"、"从此"、"不再"等词汇
   → 宣告一个新时代
   ```

3. **融入构图法**
   ```
   标题沿对角线、曲线或产品轮廓排列
   成为视觉设计的一部分
   → 文字即设计
   ```

4. **对比文案法**
   ```
   "不是...而是..."
   "告别...迎接..."
   → 强调变革和突破
   ```

**示例模板**：
```
HEADLINE TREATMENT:
- Main headline: "[宣言式中文标题]"
- Tone: [Declarative/Breakthrough/Revolutionary]
- Key numbers rendered [EXTRA LARGE/integrated into visual]
- Position: [integrated with composition, not overlaid]
- Typography: Hanken Grotesk, bold, brand color #a36710 for emphasis
```

### 3.4 PRODUCT PRESENTATION（产品展示）

**设计原则**：产品是"明星登场"，要有仪式感

**常用套路**：

1. **英雄式展示**
   ```
   产品位于画面中心，占据主导地位
   光线、构图都服务于产品展示
   → 产品是绝对主角
   ```

2. **动态登场**
   ```
   产品处于动态状态（倾斜、悬浮、旋转）
   暗示"刚刚到来"或"正在行动"
   → 能量感和新鲜感
   ```

3. **功能揭示**
   ```
   展示产品内部结构或工作状态
   （透视图、剖面图、功能可视化）
   → 技术创新的直观展示
   ```

4. **场景对比**
   ```
   产品在"问题场景"和"解决场景"中对比出现
   → 产品是变革的关键
   ```

**示例模板**：
```
PRODUCT PRESENTATION:
- Hero treatment: Product as the [star/solution/breakthrough]
- Dynamic state: [tilted/floating/in-action]
- Reveal element: [internal structure/function visualization]
- Lighting: [dramatic/spotlight/contrast]
- Product appears as if it just [arrived/solved/transformed]
```

### 3.5 CALL TO ACTION（行动号召）

**设计原则**：CTA 要有紧迫感，呼应"公告"的时效性

**常用套路**：

1. **发现式 CTA**
   ```
   "发现更多"、"探索创新"
   → 邀请用户了解这个"新事物"
   ```

2. **行动式 CTA**
   ```
   "立即体验"、"马上拥有"
   → 强调"现在就行动"
   ```

3. **限时暗示**
   ```
   虽然不直接说"限时"，但设计上暗示紧迫
   （按钮颜色醒目、位置突出）
   → 创造行动紧迫感
   ```

**示例模板**：
```
CALL TO ACTION:
- Text: "[行动式中文CTA]"
- Tone: [Discovery/Action/Urgency]
- Design: Prominent button in brand color #a36710
- Position: [bottom center/integrated with composition]
- Creates sense of: "Don't miss this"
```

### 3.6 VISUAL DISTINCTIVENESS（视觉独特性）

**设计原则**：创造"事件级"的视觉记忆

**常用套路**：

1. **色彩冲击**
   ```
   大胆的色彩对比
   品牌色作为强调色大面积使用
   → 视觉上的"大事件"感
   ```

2. **构图颠覆**
   ```
   打破品类常规的构图方式
   （对角线、分割、融合）
   → "这不是普通广告"
   ```

3. **符号创造**
   ```
   创造独特的视觉符号
   （温度计文字、时间可视化）
   → 品牌/产品的视觉签名
   ```

**示例模板**：
```
VISUAL DISTINCTIVENESS:
- Impact element: [specific visual that creates "event" feeling]
- Color strategy: [bold contrast/brand color dominance]
- Composition break: [how it differs from typical category ads]
- Memorable symbol: [unique visual element viewers will remember]
```

## 六、与其他 策略 的选择指南

| 场景 | 推荐 策略 |
|------|-------------|
| 新品上市 | **Announcement** |
| 功能升级 | **Announcement** 或 Features-Benefits |
| 日常推广 | Features-Benefits 或 Lifestyle |
| 品牌建设 | Lifestyle |
| 限时活动 | **Announcement** |
| 技术突破 | **Announcement** |

---

## 七、避免的常见错误

1. **能量不足** → Announcement 需要高能量视觉，不能太"安静"
2. **缺乏紧迫感** → 要让用户感觉"这是重要的事"
3. **产品不够突出** → 产品必须是绝对主角
4. **标题太平淡** → 需要宣言式、有力的文案
5. **与日常广告无异** → 必须有"事件感"，不是普通产品展示
6. **CTA 弱化** → Announcement 通常需要强转化引导
