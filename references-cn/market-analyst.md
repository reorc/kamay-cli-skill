# 市场分析师 (Market Analyst)

## 简介

市场分析师是一个专业的市场分析能力，专注于电商品类研究，能够将复杂的市场数据转化为可执行的营销洞察。它会自动调用 Amazon、TikTok 等平台的数据工具，深入分析品类趋势、消费者需求、产品评价和竞争定位。

## 使用场景

当你需要了解以下问题时，可以使用市场分析师：

- **品类机会评估**：某个品类是否值得进入？增长潜力如何？
- **消费者需求分析**：目标用户有什么痛点和购买驱动因素？
- **竞品分析**：竞争对手的优劣势是什么？如何差异化定位？
- **产品评价分析 (VOC)**：用户对产品的真实评价如何？有哪些改进方向？
- **季节性趋势**：什么时候是销售旺季？促销时机如何把握？
- **定价策略**：市场价格带分布如何？存在什么价格机会？

## 核心能力

### 品类趋势分析
- 识别品类增长轨迹和新兴子品类
- 发现市场饱和信号和蓝海机会
- 分析品类竞争格局和头部玩家

### 消费者需求洞察
- 提取用户痛点和未被满足的需求
- 识别购买触发因素和决策关键点
- 分析用户画像和行为特征

### 产品 VOC 分析
- 评价情感分析（正面/负面/中性）
- 提取反复出现的投诉和赞美点
- 识别星级评分的驱动因素
- 从买家视角进行竞品对比

### 季节性模式识别
- 识别销售高峰期和低谷期
- 推荐促销窗口和内容发布时机
- 分析历史趋势预测未来走势

### 定位机会挖掘
- 发现价格空白带和市场缝隙
- 识别服务不足的细分人群
- 提供差异化定位角度建议

## 如何使用

### 基本用法

直接向 Kamay 提出市场分析相关的问题即可，系统会自动调用市场分析师：

```
分析 Amazon 美站无线耳机品类，帮我了解市场机会
```

```
帮我分析这款产品的用户评价，找出主要痛点和卖点
```

```
对比分析我司产品和竞品的用户反馈差异
```

### 分析流程

1. **提出问题**：描述你想了解的市场或产品问题
2. **数据收集**：系统自动调用相关平台 API 获取数据
3. **洞察生成**：分析数据并输出营销导向的结论
4. **报告输出**：生成结构化的市场洞察报告

### 输出格式

市场分析师会生成包含以下内容的报告：

- **执行摘要**：3-5 个影响营销决策的关键发现
- **洞察分析**：品类趋势、消费者需求、VOC、季节性、定位机会
- **支持数据**：对比表格、关键指标（加粗显示）、趋势箭头（↑↓→）
- **行动建议**：优先级排序的具体建议及预期影响

## 使用提示

### 提高分析质量的建议

1. **明确分析范围**：指定平台（Amazon/TikTok）、市场（US/UK/JP 等）、品类
2. **提供产品信息**：如需 VOC 分析，提供 ASIN 或产品链接
3. **说明分析目的**：告诉分析师你的决策背景（新品开发？定价策略？广告优化？）

### 示例请求

**品类分析**：
```
分析 Amazon 美站护肤品品类，重点关注抗衰老细分市场，
我想了解市场机会和消费者需求
```

**VOC 分析**：
```
分析 ASIN B09XXXXXX 的用户评价，帮我找出产品改进方向和广告卖点
```

**竞品对比**：
```
对比以下三款产品的用户评价：B09XXXXXX, B08YYYYY, B07ZZZZZZ，
分析各自的优劣势
```
