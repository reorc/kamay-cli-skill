# Feedback 模块命令详解

## 命令总览

| 命令 | 功能 |
|------|------|
| `create` | 提交反馈 |
| `get` | 获取反馈详情 |
| `list` | 列出反馈列表 |

---

## create

向管理员提交反馈意见。

```bash
kamay feedback create \
  --title "功能建议：添加批量导出功能" \
  --content "希望能在资源列表页面添加批量导出功能，方便一次性下载多个资源文件。目前只能逐个下载，效率较低。"
```

**参数：**

| 参数 | 类型 | 必需 | 说明 |
|------|------|------|------|
| `--title` | string | 是 | 反馈标题 |
| `--content` | string | 是 | 反馈内容 |

---

## get

获取特定反馈的详细信息，包括管理员的回复。

```bash
kamay feedback get <feedback-id>
```

**返回：** 反馈详情，包括标题、内容、状态、管理员回复等

---

## list

列出已提交的反馈，可查看管理员的回复状态。

```bash
# 列出反馈
kamay feedback list

# 列出所有反馈（包括历史记录）
kamay feedback list --all
```

**参数：**

| 参数 | 类型 | 必需 | 说明 |
|------|------|------|------|
| `--all` | boolean | 否 | 显示所有反馈（包括历史记录） |
