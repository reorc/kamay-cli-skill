# 广告创意图生成器 (Ad Image Generator)

> 生成**带文字**的广告创意图，用于实际广告投放

## 简介

广告创意图生成器是一个专业的广告视觉创作工具，帮助您为产品创作高转化率的广告图片。它专注于生成**包含文案叠加**的专业广告创意，适用于电商主图、社交媒体广告、信息流广告等场景。

与氛围图生成器不同，广告图生成器**会将核心卖点文案融入视觉设计**，让广告在传递品牌调性的同时，有效传达产品价值和购买理由。

## 七种广告策略

广告创意图生成器提供 7 种经过验证的广告策略，每种策略适用于不同的营销目标。👉 [查看策略详细设计心法](./ad-image-generator/README.md)

### 1. Features-Benefits（功能优势型）

**核心理念**：用视觉语言解释"为什么这很重要"

| 特点 | 说明 |
|------|------|
| 策略核心 | 产品功能 + 用户收益 |
| 适用场景 | 功能性强的产品、技术产品 |
| 典型结构 | 功能展示 → 收益解释 → 购买理由 |

**适用产品**：电子产品、家用电器、功能性护肤品

📖 [策略详细设计心法](./ad-image-generator/features-benefits.md)

### 2. Lifestyle（生活方式型）

**核心理念**：卖的不是产品，是理想生活

| 特点 | 说明 |
|------|------|
| 策略核心 | 产品 + 理想生活场景 |
| 适用场景 | 时尚类、生活类产品 |
| 典型结构 | 生活场景 → 产品融入 → 情感共鸣 |

**适用产品**：服饰配饰、家居用品、户外装备

📖 [策略详细设计心法](./ad-image-generator/lifestyle.md)

### 3. Announcement（公告发布型）

**核心理念**：重要消息，需要被看见

| 特点 | 说明 |
|------|------|
| 策略核心 | 新品发布 + 紧迫感 |
| 适用场景 | 新品上市、重大更新 |
| 典型结构 | 发布信息 → 核心亮点 → 行动召唤 |

**适用场景**：新品首发、限量发售、品牌重大更新

📖 [策略详细设计心法](./ad-image-generator/announcement.md)

### 4. Testimonial（用户证言型）

**核心理念**：真实的声音，可信的证明

| 特点 | 说明 |
|------|------|
| 策略核心 | 真实用户好评 + 信任建立 |
| 适用场景 | 需要建立信任的产品 |
| 典型结构 | 用户证言 → 产品展示 → 信任背书 |

**适用产品**：美妆护肤、健康产品、高客单价产品

📖 [策略详细设计心法](./ad-image-generator/testimonial.md)

### 5. Problem-Solution（问题解决型）

**核心理念**：找到痛点，提供方案

| 特点 | 说明 |
|------|------|
| 策略核心 | 用户痛点 + 产品方案 |
| 适用场景 | 解决明确问题的产品 |
| 典型结构 | 痛点展示 → 产品方案 → 效果证明 |

**适用产品**：清洁用品、修复类产品、效率工具

📖 [策略详细设计心法](./ad-image-generator/problem-solution.md)

### 6. Us-vs-Them（对比竞品型）

**核心理念**：我们 vs 其他人，优势一目了然

| 特点 | 说明 |
|------|------|
| 策略核心 | 对比优势 + 差异化 |
| 适用场景 | 有明显竞争优势的产品 |
| 典型结构 | 竞品对比 → 优势展示 → 选择理由 |

**适用产品**：创新产品、性价比产品、品质升级产品

📖 [策略详细设计心法](./ad-image-generator/us-vs-them.md)

### 7. Transformation（蜕变升级型）

**核心理念**：使用前 vs 使用后，变化说话

| 特点 | 说明 |
|------|------|
| 策略核心 | 使用前对比 + 使用后效果 |
| 适用场景 | 效果可视化产品 |
| 典型结构 | 使用前状态 → 使用后变化 → 转化证明 |

**适用产品**：美妆产品、健身产品、家居改造

📖 [策略详细设计心法](./ad-image-generator/transformation.md)

## 输入要求

使用广告创意图生成器时，请提供以下信息：

### 必需信息

| 项目 | 说明 | 示例 |
|------|------|------|
| 产品描述 | 产品的核心特征和卖点 | "一款极简设计的保温杯，12小时保温，316不锈钢" |
| 产品图片 | 清晰的产品照片或渲染图 | 上传产品图片文件 |
| 广告策略 | 选择广告创意策略 | "Testimonial" 或 "用户证言型" |

### 推荐信息

| 项目 | 说明 | 示例 |
|------|------|------|
| 目标受众 | 产品面向的人群 | "25-35岁都市白领女性" |
| 核心卖点 | 最想传达的 1-3 个卖点 | "12小时保温、极简设计、便携" |
| 广告文案 | 希望呈现的文案内容 | "我的茶，从早到晚都完美" |
| 投放渠道 | 广告投放平台 | "小红书、朋友圈" |

### 渠道适配

不同渠道有不同尺寸要求：

| 渠道 | 推荐比例 | 推荐分辨率 |
|------|----------|------------|
| 淘宝主图 | 1:1 | 800×800px |
| 小红书 | 3:4 | 1080×1440px |
| 朋友圈 | 1:1 或 4:5 | 1080×1080px |
| Instagram | 1:1 或 4:5 | 1080×1350px |
| 信息流广告 | 16:9 | 1200×675px |

## 输出说明

### 输出内容

1. **完整广告创意图**：包含产品、场景、文案的成品广告图
2. **文案设计**：专业排版的广告文案
3. **行动召唤**：清晰的 CTA（Call-to-Action）引导

### 输出规格

- **分辨率**：根据投放渠道自动适配
- **格式**：标准图片格式（PNG/JPEG）
- **文案排版**：符合品牌调性的专业排版

### 输出变体

每种策略支持生成多个变体：

| 变体类型 | 说明 |
|----------|------|
| 文案变体 | 不同文案表达方式 |
| 色彩变体 | 不同的主色调 |
| 布局变体 | 不同的视觉构图 |

## 使用示例

### 示例 1：保温杯 - Testimonial 策略

**输入：**
```
产品：极简保温杯，12小时保温
策略：Testimonial（用户证言型）
文案："我的茶，从早到晚都完美"
目标用户：都市白领
```

**输出：**
- 产品在真实办公场景中
- 用户证言文案自然融入设计
- 5 星评分等信任元素
- 温暖真实的视觉氛围

### 示例 2：吸尘器 - Problem-Solution 策略

**输入：**
```
产品：无线吸尘器，60分钟续航
策略：Problem-Solution（问题解决型）
痛点：有线吸尘器受插座限制
解决方案：无线自由，全屋清洁
```

**输出：**
- 展示传统有线吸尘器的困扰
- 产品解决痛点的直观展示
- 清晰的对比和收益说明
- 强烈的行为召唤

### 示例 3：护肤品 - Transformation 策略

**输入：**
```
产品：抗衰老精华液，28天见效
策略：Transformation（蜕变升级型）
使用前：肌肤暗沉、细纹明显
使用后：肌肤焕亮、细纹淡化
```

**输出：**
- 左右对比布局
- 视觉化的效果对比
- 28天时间线展示
- 增强信任感的视觉元素

### 示例 4：耳机 - Features-Benefits 策略

**输入：**
```
产品：主动降噪耳机，40小时续航
策略：Features-Benefits（功能优势型）
功能：主动降噪
收益：在任何地方专注
```

**输出：**
- 产品功能的技术展示
- 功能带来的生活收益可视化
- 简洁的信息层级
- 科技感与生活感的平衡

## 与氛围图生成器的区别

| 特性 | 广告图生成器 | 氛围图生成器 |
|------|-------------|-------------|
| 文字内容 | 包含广告文案 | 无文字 |
| 主要用途 | 广告投放、转化导向 | 品牌探索、视觉素材 |
| 设计重点 | 信息传递、转化引导 | 视觉美感、氛围营造 |
| 策略选择 | 7 种广告策略可选 | 4 种视觉变体 |
| 适用渠道 | 广告平台、电商主图 | 详情页、社交媒体 |

## 最佳实践

### 文案撰写
- 简洁有力，一句话说清楚价值
- 使用用户语言，避免专业术语
- 包含明确的行动召唤

### 策略选择
- 根据产品特点选择合适策略
- 可以同时尝试多种策略测试效果
- 根据投放数据调整策略

### 视觉设计
- 保持品牌一致性
- 突出核心卖点
- 文案与视觉协调统一
