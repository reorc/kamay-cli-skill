# 创意简报 (Creative Brief)

## 简介

创意简报是一个将市场研究发现转化为结构化创意策略文档的能力。它能够综合市场洞察、竞品分析、品牌信息，定义清晰的创意方向，为后续的内容创作提供指导。

## 使用场景

- **广告策划**：在制作广告素材前，需要明确创意方向和核心信息
- **新品上市**：为新产品的营销活动制定创意策略
- **品牌传播**：规划品牌宣传的内容方向和调性
- **活动策划**：为促销活动或营销活动定义创意主题
- **研究落地**：将市场研究结论转化为可执行的创意指导

## 核心能力

### 研究整合
从多个来源综合信息：
- **市场洞察**：品类趋势、消费者需求、定位机会
- **竞品分析**：创意模式、有效策略、差异化空白
- **品牌档案**：品牌身份、调性、视觉规范
- **产品档案**：功能特点、独特卖点、目标受众

### 结构化输出
生成包含以下核心要素的创意简报：

| 要素 | 说明 |
|------|------|
| 目标 | 广告目标（认知/转化/互动等） |
| 目标受众 | 人口统计、心理特征、痛点 |
| 核心信息 | 希望受众记住的最重要的信息 |
| 视觉方向 | 情绪、风格、配色、参考 |
| 渠道策略 | 投放平台及平台特性考量 |
| 关键约束 | 预算、时间、品牌规范、必含元素 |
| 研究基础 | 简报所依据的研究来源 |

### 创意策略推荐
根据产品和市场情况，推荐合适的创意策略：
- 功能-利益型
- 生活方式型
- 问题-解决方案型
- 情感共鸣型

### 迭代修改
支持基于反馈的迭代优化：
- 更新目标受众
- 调整核心信息
- 修改视觉方向
- 增加或调整约束条件

## 如何使用

### 基本用法

当你完成了市场研究，需要制定创意方向时：

```
根据刚才的市场分析，帮我生成一份创意简报
```

```
为这个新品上市活动制定创意策略
```

```
生成一份创意简报，用于 Q1 春季促销活动
```

### 使用流程

1. **完成研究**：先进行市场分析、竞品研究或产品分析
2. **发起请求**：告诉 Kamay 你需要创意简报
3. **提供方向**：如有特定偏好或约束，请说明
4. **审阅修改**：查看生成的简报，提供反馈进行迭代
5. **确认定稿**：简报确认后可用于指导创作

### 输出格式

创意简报以 Markdown 格式输出，包含：

```markdown
# Creative Brief: {项目/产品名称}

## Objective
广告目标说明

## Target Audience
目标受众描述

## Core Message
核心信息及广告文案方向

## Visual Direction
视觉方向和推荐创意策略

## Channel Strategy
投放渠道及语言策略

## Key Constraints
关键约束条件

## Research Foundation
研究来源说明
```

### 语言策略

- **简报文档**：使用用户的对话语言
- **创意内容**：默认使用目标市场语言（跨境电商默认英语）

## 迭代修改

创意简报支持多轮迭代：

```
把目标受众改成 Z 世代年轻人
```

```
调性更活泼一些，增加社交媒体玩法
```

```
预算限制在 10 万以内
```

每次修改后，系统会：
1. 更新相关部分
2. 保持整体结构完整
3. 重新保存并返回最新版本

## 使用提示

### 提高简报质量的建议

1. **先完成研究**：有市场洞察和竞品分析的支持，简报更有说服力
2. **明确目标**：清楚说明广告的主要目的（品牌认知？销售转化？）
3. **描述受众**：尽可能具体地描述目标用户
4. **提供约束**：告知预算、时间、品牌规范等限制条件

### 示例请求

**完整流程**：
```
1. 先分析耳机品类的市场机会和消费者需求
2. 然后基于分析结果生成创意简报
3. 目标是新品上市的品牌认知广告
```

**快速生成**：
```
为我的护肤品牌生成创意简报，主打抗衰老功效，
目标用户是 30-45 岁的城市职业女性
```

**迭代修改**：
```
把核心信息改成强调"科学配方"而不是"天然成分"
```

## 输出交付

创意简报会自动保存并返回资源链接：
- 自动生成 Markdown 文件
- 自动上传并返回可访问链接
- 无需手动确认保存
