# Ad Image Generator

> Generate ad creative images **with text** for actual ad placements

## Introduction

Ad Image Generator is a professional ad creative tool that helps you create high-converting ad images. It focuses on generating professional ad creatives **with copy overlay**, suitable for e-commerce main images, social media ads, feed ads, and more.

Unlike Mood Image Generator, Ad Image Generator **integrates core selling copy into visual design**, delivering product value and purchase reasons while conveying brand tone.

## Seven Ad Strategies

Ad Image Generator provides 7 proven ad strategies, each suited for different marketing objectives. See [Strategy Design Guides](./ad-image-generator/README.md)

### 1. Features-Benefits

**Core Concept**: Explain "why this matters" through visual language

| Feature | Description |
|---------|-------------|
| Strategy Core | Product features + user benefits |
| Suitable For | Functional products, tech products |
| Typical Structure | Feature showcase → Benefit explanation → Purchase reason |

**Suitable Products**: Electronics, home appliances, functional skincare

See [Strategy Design Guide](./ad-image-generator/features-benefits.md)

### 2. Lifestyle

**Core Concept**: Not selling product, selling ideal lifestyle

| Feature | Description |
|---------|-------------|
| Strategy Core | Product + ideal lifestyle scene |
| Suitable For | Fashion, lifestyle products |
| Typical Structure | Lifestyle scene → Product integration → Emotional resonance |

**Suitable Products**: Fashion accessories, home goods, outdoor gear

See [Strategy Design Guide](./ad-image-generator/lifestyle.md)

### 3. Announcement

**Core Concept**: Important news, needs to be seen

| Feature | Description |
|---------|-------------|
| Strategy Core | New product launch + urgency |
| Suitable For | New launches, major updates |
| Typical Structure | Launch info → Key highlights → Call to action |

**Suitable For**: New product launches, limited editions, major brand updates

See [Strategy Design Guide](./ad-image-generator/announcement.md)

### 4. Testimonial

**Core Concept**: Real voices, credible proof

| Feature | Description |
|---------|-------------|
| Strategy Core | Real user reviews + trust building |
| Suitable For | Products needing trust |
| Typical Structure | User testimonial → Product showcase → Trust endorsement |

**Suitable Products**: Beauty/skincare, health products, high-value products

See [Strategy Design Guide](./ad-image-generator/testimonial.md)

### 5. Problem-Solution

**Core Concept**: Find pain point, provide solution

| Feature | Description |
|---------|-------------|
| Strategy Core | User pain point + product solution |

**Core Concept**: Us vs others, clear advantages

| Feature | Description |
|---------|-------------|

| Typical Structure | Competitor comparison → Advantage showcase → Selection reason |

**Suitable Products**: Innovative products, value-for-money products, quality upgrade products

**Core Concept**: Before vs after, change speaks

|---------|-------------|

## Input Requirements

When using Ad Image Generator, please provide:

### Required Information

| Item | Description | Example |
|------|-------------|---------|
| Product Description | Core features and selling points | "A minimalist thermos, 12-hour insulation, 316 stainless steel" |
| Product Images | Clear product photos or renderings | Upload product image files |
| Ad Strategy | Choose ad creative strategy | "Testimonial" or "User testimonial type" |

### Recommended Information

| Item | Description | Example |
|------|-------------|---------|
| Target Audience | People the product targets | "Urban white-collar women aged 25-35" |
| Core Selling Points | 1-3 points you want to convey most | "12-hour insulation, minimalist design, portable" |
| Ad Copy | Copy you want to present | "My tea, perfect from morning to night" |
| Placement Channels | Ad placement platforms | "Xiaohongshu, WeChat Moments" |

### Channel Adaptation

Different channels have different size requirements

| Channel | Recommended Ratio | Recommended Resolution |
|---------|-------------------|------------------------|
| Taobao Main Image | 1:1 | 800x800px |
| Xiaohongshu | 3:4 | 1080x1440px |
| WeChat Moments | 1:1 or 4:5 | 1080x1080px |
| Instagram | 1:1 or 4:5 | 1080x1350px |
| Feed Ads | 16:9 | 1200x675px |

## Output

### Output Content

1. **Complete Ad Creative Images**: Finished ad images including product, scene, copy
2. **Copy Design**: Professionally typeset ad copy
3. **Call to Action**: Clear CTA (Call-to-Action) guidance

### Output Specifications
- **Resolution**: Automatically adapted based on placement channel
- **Format**: Standard image formats (PNG/JPEG)

| Variant Type | Description |
|--------------|-------------|

## Usage Examples

### Example 1: Thermos - Testimonial Strategy
**Input:**
```
Product: Minimalist thermos, 12-hour insulation
Strategy: Testimonial (User testimonial type)
Copy: "My tea, perfect from morning to night"
Target User: Urban white-collar workers
```
**Output:**
- Product in real office scene
- User testimonial copy naturally integrated into design
- 5-star rating and other trust elements
- Warm, authentic visual atmosphere

### Example 2: Vacuum - Problem-Solution Strategy
**Input:**
```
Product: Cordless vacuum, 60-minute runtime
Strategy: Problem-Solution (Problem-solving type)
Pain Point: Corded vacuums limited by outlets
Solution: Cordless freedom, whole-house cleaning
```
**Output:**
- Show traditional corded vacuum frustration
- Intuitive product pain point solution showcase
- Clear comparison and benefit explanation
- Strong call to action

### Example 3: Skincare - Transformation Strategy
**Input:**
```
Product: Anti-aging serum, 28-day visible results
Strategy: Transformation (Upgrade type)
Before: Dull skin, visible fine lines
After: Bright skin, diminished fine lines
```
**Output:**
- Before/after comparison layout
- Visual effect comparison
- 28-day timeline display
- Trust-enhancing visual elements

### Example 4: Headphones - Features-Benefits Strategy
**Input:**
```
Product: Active noise-canceling headphones, 40-hour battery life
Strategy: Features-Benefits (Feature advantage type)
Feature: Active noise cancellation
Benefit: Focus anywhere
```
**Output:**
- Product feature technical showcase
- Feature benefit lifestyle visualization
- Clear information hierarchy
- Balance between tech feel and lifestyle feel

## Best Practices

### Copy Writing
- Concise and powerful, one sentence to clarify value
- Use user language, avoid technical jargon
- Include clear call to action

### Strategy Selection
- Choose appropriate strategy based on product characteristics
- Can test multiple strategies simultaneously
- Adjust strategy based on placement data

### Visual Design
- Maintain brand consistency
- Highlight core selling points
- Coordinate copy and visuals
