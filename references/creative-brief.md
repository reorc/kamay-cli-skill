# Creative Brief

## Introduction

Creative Brief is a capability that transforms market research findings into structured creative strategy documents. It synthesizes market insights, competitive analysis, brand information, and defines clear creative direction to guide subsequent content creation.

## Use Cases

- **Ad Planning**: Need to clarify creative direction and core messages before producing ad materials
- **New Product Launch**: Define creative strategy for new product marketing campaigns
- **Brand Communication**: Plan content direction and tone for brand promotion
- **Campaign Planning**: Define creative themes for promotional or marketing campaigns
- **Research Activation**: Transform market research conclusions into actionable creative guidance

## Core Capabilities

### Research Synthesis
Synthesize information from multiple sources:
- **Market Insights**: Category trends, consumer demand, positioning opportunities
- **Competitive Analysis**: Creative patterns, effective tactics, differentiation gaps
- **Brand Profile**: Brand identity, tone, visual guidelines
- **Product Profile**: Features, USPs, target audience

### Structured Output
Generate creative briefs containing these core elements:

| Element | Description |
|---------|-------------|
| Objective | Ad goal (awareness/conversion/engagement, etc.) |
| Target Audience | Demographics, psychographics, pain points |
| Core Message | The most important thing you want audience to remember |
| Visual Direction | Mood, style, color palette, references |
| Channel Strategy | Placement platforms and platform-specific considerations |
| Key Constraints | Budget, timeline, brand guidelines, mandatory elements |
| Research Foundation | Research sources the brief is based on |

### Creative Strategy Recommendations
Recommend suitable creative strategies based on product and market situation:
- Features-Benefits
- Lifestyle
- Problem-Solution
- Emotional resonance

### Iteration Support
Support iteration based on feedback:
- Update target audience
- Adjust core message
- Modify visual direction
- Add or adjust constraints

## How to Use

### Basic Usage

When you've completed market research and need to define creative direction:

```
Based on the market analysis just now, help me generate a creative brief
```

```
Define creative strategy for this new product launch campaign
```

```
Generate a creative brief for the Q1 spring promotion campaign
```

### Workflow

1. **Complete Research**: First conduct market analysis, competitive research, or product analysis
2. **Initiate Request**: Tell Kamay you need a creative brief
3. **Provide Direction**: If you have specific preferences or constraints, mention them
4. **Review & Revise**: Review the generated brief, provide feedback for iteration
5. **Confirm Final**: After confirmation, brief can guide creative work

### Output Format

Creative brief is output in Markdown format, containing:

```markdown
# Creative Brief: {Project/Product Name}

## Objective
Ad goal description

## Target Audience
Target audience description

## Core Message
Core message and ad copy direction

## Visual Direction
Visual direction and recommended creative strategies

## Channel Strategy
Placement channels and language strategy

## Key Constraints
Key constraint conditions

## Research Foundation
Research source description
```

### Language Strategy

- **Brief Document**: Uses user's conversation language
- **Creative Content**: Defaults to target market language (cross-border e-commerce defaults to English)

## Iteration

Creative brief supports multiple rounds of iteration:

```
Change the target audience to Gen Z
```

```
Make the tone more playful, add social media engagement elements
```

```
Budget constraint within 100k
```

After each revision, the system will:
1. Update relevant sections
2. Maintain overall structure integrity
3. Re-save and return latest version

## Usage Tips

### Tips to Improve Brief Quality

1. **Complete Research First**: With market insight and competitive analysis support, briefs are more persuasive
2. **Clarify Objectives**: Clearly state the main ad purpose (brand awareness? sales conversion?)
3. **Describe Audience**: Describe target users as specifically as possible
4. **Provide Constraints**: Inform about budget, timeline, brand guidelines, and other limitations

### Example Requests

**Complete Workflow**:
```
1. First analyze the earphones category market opportunities and consumer demand
2. Then generate a creative brief based on the analysis results
3. Goal is brand awareness advertising for new product launch
```

**Quick Generation**:
```
Generate a creative brief for my skincare brand, focusing on anti-aging benefits,
target audience is 30-45 year old urban professional women
```

**Iteration**:
```
Change the core message to emphasize "scientific formulation" instead of "natural ingredients"
```
