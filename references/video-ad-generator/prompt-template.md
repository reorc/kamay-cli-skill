# 电商视频 Prompt 模板

> 基于八大爆款逻辑，适用于 Kling / Veo / Sora-2 / Seedance 等视频生成模型

## 爆款逻辑速查

| 逻辑 | 适用场景 | 画面表现 |
|-----|---------|---------|
| 成本 | 强调性价比/省钱 | 突出"平价感"或"对比感" |
| 人群 | 精准定位群体 | 体现特定人群使用场景 |
| 猎奇 | 新奇/罕见 | "没见过"的视觉冲击 |
| 反差 | 打破认知 | 前后对比/意外转折 |
| 最差 | 避坑/负面 | 展示"错误示范"再给正解 |
| 头牌 | 最好/顶级 | 强调品质感/仪式感 |
| 怀旧 | 情感共鸣 | 营造年代感/回忆氛围 |
| 荷尔蒙 | 两性吸引 | 突出身材/魅力展示 |

## 视频模型约束速查

选模型时必须参考此表，避免使用不支持的参数。

| 模型 | 时长 | 比例 | 图片能力 | 特殊限制 |
|------|------|------|---------|---------|
| Kling-V3 | 3-15s 灵活 | 16:9, 9:16, 1:1 | 首帧 image_start, 尾帧 image_end | 提供 image_start 自动切换 i2v 模式 |
| Veo 3.1 | 4/6/8s 固定 | 16:9, 9:16 | 首帧, 尾帧, style_reference, subject_reference | **reference image 时强制 16:9**; subject_reference 强制 8s |
| Sora-2 | 4/8/12s 固定 | 16:9, 9:16 | 最多 1 张参考图 | **不支持人脸参考** |
| Seedance 1.5 | 4-12s 灵活 | 16:9, 4:3, 1:1, 3:4, 9:16, 21:9 | 首帧, 尾帧, 1-4 张参考图 | draft 模式 480p 预览; 首帧+尾帧互斥参考图模式 |

### 模型选择策略

| 镜头场景 | 推荐模型 | 图片使用方式 | 理由 |
|---------|---------|------------|------|
| 产品特写/高还原 | Veo | subject_reference（产品原图）+ style_reference（参考图） | subject_reference 保产品一致性最强 |
| 人物+产品交互 | Kling | 先出参考图 → image_start | 灵活时长，人物生成稳定 |
| 纯氛围/运镜 | Sora-2 / Kling | text-to-video 或参考图做首帧 | 简单镜头不需要复杂参考 |
| 首尾帧过渡 | Veo / Seedance | first_frame + last_frame | 支持帧间插值 |
| 无人脸场景 | Sora-2 | 参考图 | 画质好但不支持人脸参考 |
| 多比例需求 | Seedance | 参考图 | 比例最灵活（6 种） |

## 风格锁定机制（Style Prefix）

用户选定方案后，提取一组风格关键词作为 **Style Prefix**，后续所有生图和视频 prompt 都以此为前缀。

Style Prefix 必须包含：
- **色调/调色板**（如 warm golden tones, sakura pink and white palette）
- **光线风格**（如 soft natural light, dramatic side lighting）
- **整体美学**（如 minimal lifestyle, industrial chic, editorial fashion）
- **质感/氛围**（如 cinematic, dreamy bokeh, clean and crisp）

示例：
```
Warm golden hour lighting, soft bokeh background,
minimal lifestyle aesthetic, sakura pink and white
color palette, natural indoor setting, cinematic grain
```

## 分镜 Prompt 规范

### 字数限制（CRITICAL）

| 语言 | 单镜头 Prompt 上限 | 说明 |
|------|-------------------|------|
| 英文 | 800 characters | 视频模型输入限制 |
| 中文 | 500 字符 | 视频模型输入限制 |

### Prompt 结构

每个 Shot 的视频 prompt 格式：

```
[Style Prefix]

[Shot Description]
- Camera: [景别 + 运镜]
- Scene: [核心画面描述]
- Product: [产品状态]
- Motion: [关键运动/动作]
```

### 精简原则

- 以 Style Prefix 开头，不在 shot 描述中重复风格信息
- 聚焦：主体 + 动作 + 运镜 + 光线
- 删除冗余形容词
- 音频描述单独列出，不计入视频 prompt 字数（音频通过模型 sound_effects 参数或后期处理）

## 视觉锚点机制

**视觉锚点是保证跨分镜一致性的核心机制。** 在写分镜脚本之前，先生成一组"锚点图"，作为后续所有生图和生视频的视觉参考。

### 锚点类型

| 锚点类型 | 何时需要 | 生成要求 |
|---------|---------|---------|
| 产品锚点 | 产品出现 ≥2 个镜头 | 产品在目标场景中的标准照；**必须用产品原图做 reference_images** |
| 角色锚点 | 有人物 ≥2 个镜头 | 确定外貌/穿着/发型的标准照；prompt 中明确描述特征 |
| 场景锚点 | 多镜头共用同一空间 | 建立空间布局、光线、道具位置的基调 |

### 锚点按视频类型的典型组合

| 视频类型 | 需要的锚点 |
|---------|-----------|
| 纯产品展示片（无人物） | 产品锚点 + 场景锚点 |
| UGC 口播/场景故事片 | 角色锚点 + 产品锚点 + 场景锚点 |
| 抽象品牌片 | 场景锚点（可能不需要其他锚点） |
| 对比/开箱 | 角色锚点 + 产品锚点 |

### 锚点引用规则

每个 shot 标注"引用锚点"字段：

| 镜头内容 | 引用的锚点 |
|---------|-----------|
| 产品出现（特写/背景均算） | 产品锚点 |
| 人物出现 | 角色锚点 |
| 在主场景中 | 场景锚点 |
| 纯氛围（无产品无人物，或全新场景） | **不引用任何锚点**，直接 text-to-video |

### 锚点在视频生成中的使用方式

| 模型 | 使用方式 |
|------|---------|
| Veo | subject_reference = 产品原图；style_reference = 锚点图 or shot 参考图 |
| Kling | image_start = 锚点图 or shot 参考图 |
| Sora-2 | image_url = 锚点图（最多 1 张；不支持人脸） |
| Seedance | reference_image = 多个锚点图（最多 4 张） |

### 角色一致性（CRITICAL）

当视频包含人物时，角色一致性是最大挑战：

1. **角色锚点图**必须明确描述：肤色、发型、发色、体型、穿着、年龄范围
2. **所有人物镜头的 prompt** 都必须复用这段外貌描述（逐字复用，不要换措辞）
3. **所有人物镜头的参考图/首帧**都必须以角色锚点作为 reference_images 生成
4. 如果某个 shot 的画面与角色锚点差异较大（不同动作/构图），先用 generate_image 基于角色锚点生成该 shot 的专属参考图，再用参考图做视频首帧

## 分镜数量控制

分镜越多，跨镜头一致性风险越大。**优先使用更少但更长的镜头。**

| 视频时长 | 推荐 shot 数 | 每 shot 时长 |
|---------|-------------|-------------|
| 5-8s | 2-3 个 | 3-4s |
| 10-15s | 3-4 个 | 3-5s |
| 30s | 4-6 个 | 4-6s |

## Prompt 输出结构

最终保存的创意文档中，每个 Shot 的完整信息：

```
[Shot N: Xs-Ys — Title]

景别/运镜：[描述]
画面：[描述]
产品状态：[描述]
音频：[音乐 + 音效 + 画外音]
引用锚点：[产品/角色/场景/无]
推荐模型：[模型名]

视频 Prompt：
"""
[Style Prefix]
[精简的 shot 描述，不超过字数限制]
"""
```

## 音频设计指南

### 背景音乐设计原则

1. 音乐必须服务于情绪弧线（如：紧张→转折→自信→收尾）
2. 每个时间段标注 BPM，确保节奏与画面匹配
3. 转折点的音乐变化要与画面转折同步
4. 收尾留白，不要音乐突然截断

### 音效设计原则

1. 每个关键动作都需要对应音效
2. 音效必须精确到秒，与画面同步
3. 音效层次：前景音效 > 背景音效 > 环境音
4. 转折点需要标志性音效作为记忆锚点

### 画外音设计原则

画外音密度取决于视频类型：

**品牌广告片/场景故事片**（少量点睛）：
1. 文案要短，15s 视频 2-3 句话足够
2. 留出空白给音效和画面呼吸

**UGC 口播/测评**（持续讲解）：
1. 几乎全程在说话，需要完整台词脚本
2. 逐句标注时间段和情绪语气
3. 背景音乐压低，不抢人声
