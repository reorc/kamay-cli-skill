# Us-vs-Them 广告 Prompt 架构设计心法

> 核心理念：**通过对比竞品，凸显自身优势**

## 一、策略 定位

Us-vs-Them 类型广告的核心目标是**通过与竞品或旧方案的直接对比，让用户清晰看到选择你的理由**。这是最具攻击性的广告类型，需要谨慎但有力地执行。

**关键心智模型**：
- "他们" → 旧方案/竞品的劣势
- "我们" → 新方案/产品的优势
- 对比 → 选择的理由

**策略定位**：
```
不是"我们很好"，而是"我们比他们更好"
不是"功能介绍"，而是"优势对比"
```

## 三、章节设计套路

### 3.1 COMPOSITION & LAYOUT（构图与布局）

**设计原则**：用空间分割创造"对抗感"

**常用套路**：

1. **垂直分割（经典）**
   ```
   左侧：旧方案/竞品（问题）
   中心：分割线（对抗）
   右侧：我方产品（解决）
   → 直观的左右对比
   ```

2. **对角线分割（创新）**
   ```
   从左上到右下的对角线分割
   创造动态能量和视觉张力
   → 打破传统对比广告的静态感
   ```

3. **中心爆发式**
   ```
   产品位于中心
   旧方案元素在周围"被击退"
   → 产品是"胜利者"
   ```

**示例模板**：
```
COMPOSITION & LAYOUT:
- Split type: [Vertical/Diagonal/Center-burst]
- Division ratio: [50-50 / 40-60 / Dynamic]
- Visual tension: [how the two sides "confront" each other]
- Product position: [center divider / right side hero]
```

### 3.2 LEFT SIDE - "THEM"（他们/旧方案）

**设计原则**：展示"不选择我们"的代价，但不要过度贬低

**视觉化"他们"的套路**：

1. **复杂/混乱**
   ```
   多个物品散乱（旧保温杯、清洁剂、毛巾、刷子）
   视觉上的"杂乱"和"麻烦"
   → "旧方案很复杂"
   ```

2. **褪色/陈旧**
   ```
   灰暗、褪色的色调
   暗示"过时"和"落后"
   → "旧方案已经过时"
   ```

3. **问题可视化**
   ```
   温度下降的视觉暗示（冷凝、蒸汽消失）
   细菌/霉菌的暗示（不直接展示）
   → "旧方案有问题"
   ```

4. **标签化**
   ```
   "旧的麻烦" / "OLD WAY" / "传统方案"
   简洁的标签定义这一侧
   ```

**示例模板**：
```
LEFT SIDE - "THEM" (Old Way / Competitors):
- Visual representation: [cluttered/faded/problematic]
- Color tone: [muted/grey/warm-but-tired]
- Elements: [specific items representing old solution]
- Label: "[旧的麻烦]" or similar
- Emotion evoked: [frustration/complexity/outdated]
- Note: Show problems, but don't mock or disrespect
```

### 3.3 RIGHT SIDE - "US"（我们/新方案）

**设计原则**：展示选择我们的"奖励"

**视觉化"我们"的套路**：

1. **简洁/有序**
   ```
   只有产品，干净整洁
   视觉上的"简单"和"优雅"
   → "我们的方案很简单"
   ```

2. **明亮/现代**
   ```
   明亮、清新的色调
   暗示"先进"和"创新"
   → "我们的方案很现代"
   ```

3. **功能可视化**
   ```
   产品在理想状态下工作
   蒸汽稳定、温度恒定
   → "我们的方案有效"
   ```

4. **生活场景**
   ```
   产品融入美好生活场景
   户外、自然光、自由感
   → "选择我们 = 更好的生活"
   ```

**示例模板**：
```
RIGHT SIDE - "US" (New Way / Our Product):
- Visual representation: [clean/bright/functional]
- Color tone: [bright/cool/modern]
- Product shown: [in ideal state/in lifestyle context]
- Label: "[新的自由]" or similar
- Emotion evoked: [relief/simplicity/aspiration]
- Benefit icons: [specific advantages visualized]
```

### 3.4 CENTER DIVIDER（中心分割线）

**设计原则**：分割线本身也是设计元素，不只是"线"

**常用套路**：

1. **能量线**
   ```
   品牌色的动态线条
   暗示"能量"和"变革"
   → 分割线是"力量"
   ```

2. **VERSUS 文字**
   ```
   "VS" 或 "对比" 融入分割线
   成为视觉焦点
   → 明确的对抗感
   ```

3. **产品作为分割**
   ```
   产品本身位于分割线位置
   成为"桥梁"或"选择"
   → 产品是决策点
   ```

4. **对角线切割**
   ```
   不是垂直线，而是对角线
   创造动态和能量
   → 打破静态对比
   ```

**示例模板**：
```
CENTER DIVIDER:
- Type: [Energy line/VERSUS text/Product as divider/Diagonal]
- Color: Brand #a36710 or [contrast color]
- Visual treatment: [glow/bold/dynamic]
- Text (optional): "VS" or "[对比]"
- Creates: [tension/choice point/transformation moment]
```

### 3.5 HEADLINE TREATMENT（标题处理）

**设计原则**：标题必须强化"对比"的核心信息

**常用文案套路**：

1. **对比式**
   ```
   "旧的麻烦 vs 新的自由"
   "复杂 vs 简单"
   → 直接的对比陈述
   ```

2. **选择式**
   ```
   "你选择哪一边？"
   "是时候升级了"
   → 引导用户做选择
   ```

3. **优势宣言式**
   ```
   "24小时，只降2度"
   "这就是区别"
   → 强调核心优势
   ```

4. **未来式**
   ```
   "未来的你已经在用"
   "聪明人的选择"
   → 暗示选择我们是"正确"的
   ```

**示例模板**：
```
HEADLINE TREATMENT:
- Main headline: "[对比式/选择式/优势式文案]"
- Position: [crossing the divider / integrated with composition]
- Typography: Hanken Grotesk, bold
- Contrast: [problem text muted] vs [solution text prominent]
- Brand color #a36710 for [key emphasis]
```

### 3.6 KEY CALLOUT BOXES（优势点对比）

**设计原则**：用简洁的对比点强化差异

**常用套路**：

1. **图标对比**
   ```
   左侧：❌ 需要清洁剂
   右侧：✅ 无需清洁剂
   → 视觉化的优劣对比
   ```

2. **数据对比**
   ```
   左侧：温度下降 10°+
   右侧：温度下降 <2°
   → 数据化的优势证明
   ```

3. **沿分割线排列**
   ```
   对比点沿着中心分割线排列
   成为视觉设计的一部分
   → 对比点融入构图
   ```

**示例模板**：
```
KEY CALLOUT BOXES (along the divider):
- Comparison 1: [旧方案问题] vs [我方优势]
- Comparison 2: [旧方案问题] vs [我方优势]
- Comparison 3: [旧方案问题] vs [我方优势]
- Visual style: [icons/checkmarks/data]
- Position: [along divider/floating/integrated]
```

### 3.7 UNEXPECTED ELEMENT（意外元素）

**设计原则**：打破传统对比广告的"无聊"

**常用套路**：

1. **对角线分割**
   ```
   不是垂直分割，而是对角线
   创造动态和能量
   → "这不是普通的对比广告"
   ```

2. **标题融入分割**
   ```
   标题沿着分割线排列
   成为视觉设计的一部分
   → 文字即设计
   ```

3. **产品"打破"分割**
   ```
   产品从"我们"这边"突破"到"他们"那边
   暗示"取代"和"胜利"
   → 产品是"赢家"
   ```

**示例模板**：
```
UNEXPECTED ELEMENT:
- Break from typical comparison ads by [specific technique]
- Creates visual [tension/energy/surprise]
- Makes viewers think: "I haven't seen comparison ads like this"
```

## 六、使用注意事项

### 何时使用 Us-vs-Them

- ✅ 产品有明确的差异化优势
- ✅ 目标用户正在比较不同方案
- ✅ 需要快速建立"为什么选我们"的认知
- ✅ 竞品或旧方案有明显的痛点

### 何时避免 Us-vs-Them

- ❌ 没有明确的差异化优势
- ❌ 可能引发法律风险（直接点名竞品）
- ❌ 目标用户还不知道有问题存在
- ❌ 品牌调性不适合"攻击性"广告

### 执行原则

1. **对比事实，不贬低竞品**
   - 展示客观差异，不使用贬义词
   - 用"旧方案"而非具体竞品名称

2. **强调"我们更好"，而非"他们很差"**
   - 重心在展示自身优势
   - 问题展示点到为止

3. **保持品牌调性**
   - 即使是对比广告，也要保持品牌的专业和尊重

---

## 七、避免的常见错误

1. **对比不够鲜明** → 两侧差异必须一眼可见
2. **过度贬低竞品** → 保持专业，避免法律风险
3. **产品不够突出** → 产品必须是"赢家"的形象
4. **分割线太弱** → 分割线本身要有设计感
5. **缺乏行动引导** → 对比后要引导用户"选择我们"
6. **视觉太静态** → 使用对角线或动态元素打破沉闷
