# Lifestyle 广告 Prompt 架构设计心法

> 核心理念：**产品是生活的配角，场景是真正的主角**

## 一、策略 定位

Lifestyle 类型广告的核心目标是**将产品融入理想生活场景**，让用户看到的不是产品，而是"拥有这个产品后的自己"。

**关键心智模型**：
- 产品 → 生活方式的符号
- 功能 → 生活品质的保障
- 场景 → 用户的理想自我投射

**与 Features-Benefits 的核心区别**：
| 维度 | Features-Benefits | Lifestyle |
|------|-------------------|-----------|
| 主角 | 产品功能 | 生活场景 |
| 情感 | 信任、可靠 | 向往、归属 |
| 叙事 | "它能做什么" | "你会成为谁" |
| 视觉 | 产品特写 | 场景氛围 |

## 三、章节设计套路

### 3.1 COMPOSITION & EMOTIONAL STORY（构图与情感叙事）

**设计原则**：用场景讲故事，产品是故事中的"可靠伙伴"

**常用套路**：

1. **24小时生活流**
   ```
   画面展示一天中的多个时刻：
   晨间准备 → 通勤路上 → 工作间隙 → 户外活动 → 傍晚放松
   产品贯穿始终，成为"恒定的陪伴"
   ```

2. **生活场景堆叠**
   ```
   垂直构图中堆叠多个生活片段
   产品位于中心，连接不同场景
   → 隐喻：无论何时何地，温度恒定
   ```

3. **氛围渐变构图**
   ```
   背景从冷色调渐变到暖色调
   或从清晨光线渐变到黄昏光线
   → 时间流逝，产品不变
   ```

4. **人物视角构图**
   ```
   第一人称视角：手持产品看向远方
   或第三人称：人物与产品的互动瞬间
   → 代入感和真实感
   ```

**示例模板**：
```
COMPOSITION & EMOTIONAL STORY:
- Build a visual narrative showing [life scenario]
- Product appears as the constant companion through [time/scenes]
- Eye flow guides viewer through: [moment 1] → [product] → [moment 2]
- Emotional arc: [starting emotion] → [ending emotion]
- The product enables/supports this lifestyle, not dominates it
```

### 3.2 EMOTIONAL STORY / MOOD（情感叙事/氛围）

**设计原则**：定义用户看到广告时应该"感受到"什么

**常用情感定位**：

1. **Aspirational（向往型）**
   ```
   关键词：理想、追求、品质生活
   视觉：金色光线、自然风光、精致细节
   → "这是我想要的生活"
   ```

2. **Warm & Approachable（温暖亲近型）**
   ```
   关键词：陪伴、日常、舒适
   视觉：柔和光线、家居场景、亲密互动
   → "这让我感到安心"
   ```

3. **Active & Dynamic（活力动感型）**
   ```
   关键词：冒险、自由、探索
   视觉：户外场景、运动瞬间、自然元素
   → "这支持我的冒险"
   ```

**示例模板**：
```
MOOD:
- Overall feeling: [Aspirational/Warm/Active]
- Emotional keywords: [3-4个情感关键词]
- Visual atmosphere: [光线、色调、场景类型]
- Viewer should feel: "[用户内心独白]"
```

### 3.3 UNEXPECTED ELEMENT（意外元素）

**设计原则**：在生活场景中创造"诗意的惊喜"

**Lifestyle 特有套路**：

1. **蒸汽文字法**
   ```
   产品内部的蒸汽形成文字或图案
   → 功能变成诗意的视觉语言
   ```

2. **时间凝固法**
   ```
   生活场景中的某个瞬间被"定格"
   产品是这个完美瞬间的一部分
   → 强调"值得珍藏的时刻"
   ```

3. **场景穿越法**
   ```
   产品出现在多个场景中，暗示它"穿越"了一天
   → 可靠的全天候陪伴
   ```

4. **视角颠覆法**
   ```
   俯视、仰视、反射等非常规视角
   → 熟悉场景的新鲜感
   ```

**示例模板**：
```
UNEXPECTED ELEMENT:
- Create a poetic visual surprise within the lifestyle context
- [Specific technique]: [具体描述]
- This transforms a mundane moment into something memorable
- The surprise reinforces the emotional narrative, not disrupts it
```

### 3.4 PRODUCT PRESENTATION - ALIVE & PURPOSEFUL（产品展示）

**设计原则**：产品必须"有目的地存在"于场景中，而非硬塞进去

**常用套路**：

1. **自然握持法**
   ```
   人手自然握持产品，进行真实活动
   （边走边喝、边工作边放在桌上）
   → 产品是生活的一部分
   ```

2. **环境融合法**
   ```
   产品与环境元素产生互动
   （阳光照射、与其他物品并置）
   → 产品属于这个场景
   ```

3. **功能暗示法**
   ```
   不直接展示功能，而是通过细节暗示
   （表面无冷凝 = 保温好、茶叶可见 = 可泡茶）
   → 功能融入场景叙事
   ```

**示例模板**：
```
PRODUCT PRESENTATION - ALIVE & PURPOSEFUL:
- Product shown in [natural use context]
- Interaction: [how product relates to scene/person]
- Subtle benefit cues: [visual hints of functionality]
- Product solves a real need in this moment: [specific need]
- NOT a product shot inserted into a scene - it BELONGS here
```

### 3.5 VISUAL DISTINCTIVENESS（视觉独特性）

**设计原则**：创造"氛围记忆"而非"产品记忆"

**常用套路**：

1. **光线记忆法**
   ```
   独特的光线处理成为视觉签名
   （金色黄昏、清晨薄雾、窗边光影）
   ```

2. **色彩情绪法**
   ```
   整体色调传递情绪
   （温暖的棕橙色调 = 舒适；清冷的蓝绿色调 = 清新）
   ```

3. **场景符号法**
   ```
   特定场景元素成为记忆锚点
   （山顶、茶园、晨间厨房）
   ```

**示例模板**：
```
VISUAL DISTINCTIVENESS:
- Memorable atmosphere: [specific visual treatment]
- Signature element: [light/color/scene that defines the mood]
- This creates an emotional imprint, not just product recall
- Viewers remember the FEELING before they remember the product
```

### 3.6 LIFESTYLE NARRATIVE（生活方式叙事）

**设计原则**：讲述"拥有这个产品的人"的故事

**叙事框架**：

1. **日常仪式型**
   ```
   展示产品如何融入日常仪式
   （晨间泡茶、午后小憩、通勤路上）
   → 产品是美好习惯的一部分
   ```

2. **人生旅程型**
   ```
   展示产品陪伴用户经历不同场景
   （工作、旅行、休闲）
   → 产品是人生旅途的伙伴
   ```

3. **品质追求型**
   ```
   展示用户对生活品质的追求
   产品是这种追求的体现
   → 产品代表一种生活态度
   ```

**示例模板**：
```
LIFESTYLE NARRATIVE:
- Story type: [Daily Ritual / Life Journey / Quality Pursuit]
- The person in this ad values: [lifestyle values]
- Product role: [how product supports/enables this lifestyle]
- Implicit message: "People who [value X] choose this product"
```

## 六、与 Features-Benefits 的对比指南

| 决策点 | 选 Features-Benefits | 选 Lifestyle |
|--------|---------------------|--------------|
| 用户需求 | "这个产品好不好用？" | "这个产品适合我吗？" |
| 产品阶段 | 新品上市、功能升级 | 品牌建设、情感连接 |
| 竞争策略 | 功能差异化 | 情感差异化 |
| 视觉重心 | 产品特写 | 场景氛围 |
| 文案风格 | 参数、承诺 | 故事、情感 |

---

## 七、避免的常见错误

1. **产品硬塞进场景** → 产品必须"属于"这个场景
2. **功能点罗列** → 用场景暗示功能，而非直接说明
3. **缺乏人物元素** → 需要人的存在来创造代入感
4. **场景过于完美** → 保持真实感，避免"广告感"
5. **情感定位模糊** → 明确是向往型、温暖型还是活力型
6. **忽略时间叙事** → Lifestyle 广告需要"时间感"
