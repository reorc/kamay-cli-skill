# Mood Image Generator

> Generate **text-free** mood/lifestyle images for brand visual exploration

## Introduction

Mood Image Generator is a professional visual creation tool that helps you create emotionally resonant visual content for products. It focuses on generating high-quality lifestyle images with **pure visuals, no text**, suitable for brand visual exploration, e-commerce detail pages, social media content, and more.

Unlike Ad Image Generator, Mood Image Generator **includes no text overlay**, allowing products to naturally present in carefully designed environments, conveying brand tone and product value through visual language.

## Use Cases

### Brand Visual Exploration
- Establish brand visual style guidelines
- Explore different visual directions and tones
- Provide creative references for design teams

### E-commerce Detail Pages
- Atmospheric product hero display
- Contextualized product images
- Lifestyle content supplements

### Social Media Content
- Instagram/Xiaohongshu style images
- Brand-consistent visual content
- Lifestyle product display

### Pre-Ad Production
- Creative concept validation
- Visual direction testing
- Reference for subsequent ad production

## Input Requirements

### Required Information

| Item | Description | Example |
|------|-------------|---------|
| Product Description | Core features and selling points of the product | "A minimalist thermos, 304 stainless steel liner, 12-hour insulation" |
| Product Images | Clear product photos or renderings | Upload product image files |

### Recommended Information

| Item | Description | Example |
|------|-------------|---------|
| Target Audience | People the product targets | "Urban white-collar women aged 25-35" |
| Brand Tone | Brand feeling to convey | "Minimalist, natural, high quality" |
| Usage Scenarios | Typical usage environments | "Office, commute, home" |
| Visual Reference | Preferred visual style references | "Japanese minimalist style, Nordic natural style" |

## Output

### Output Content

1. **High-quality Mood Images**: Carefully composed product scenes, no text overlay
2. **Multiple Angles**: Product details and environmental atmosphere
3. **Professional Lighting**: Lighting effects meeting high-end commercial photography standards

### Output Specifications

- **Resolution**: High-resolution output, suitable for multiple uses
- **Format**: Standard image formats (PNG/JPEG)
- **Aspect Ratio**: Adjustable as needed (1:1, 4:5, 16:9, etc.)

### Variant Types

Mood Image Generator provides 4 variant strategies:

| Variant Type | Description | Use Case |
|--------------|-------------|----------|
| Scene Variant | Different backgrounds and environments | Test different usage scenarios |
| Human Variant | Adjust human elements | Show target user groups |
| Angle Variant | Different shooting angles | Get multi-angle assets |
| Lighting Variant | Different lighting effects | Explore different mood atmospheres |

## Usage Examples

### Example 1: Minimalist Thermos

**Input:**
```
Product: Minimalist thermos, 304 stainless steel liner, 12-hour insulation
Target User: Urban white-collar workers aged 25-35
Brand Tone: Minimalist, natural, high quality
Usage Scenario: Office, commute
```
**Output:**
- Product in naturally lit office desk scene
- Paired with notebook, plants, and other minimalist elements
- Warm morning light atmosphere

### Example 2: Outdoor Backpack
**Input:**
```
Product: 30L outdoor hiking backpack, waterproof fabric, multi-compartment
Target User: Outdoor sports enthusiasts
Brand Tone: Adventurous, free, energetic
Usage Scenario: Mountains, camping
```
**Output:**
- Backpack in natural mountain environment
- Paired with hiking poles, water bottles, and other outdoor elements
- Morning or evening natural light

### Example 3: High-end Skincare
**Input:**
```
Product: Anti-aging serum, glass bottle, gold cap
Target User: Urban women aged 30-45
Brand Tone: Luxurious, elegant, tech-forward
Usage Scenario: Bathroom, vanity table
```
**Output:**
- Product on exquisite vanity table
- Paired with flowers, aromatherapy, and other elegant elements
- Soft high-end lighting

## Difference from Ad Image Generator

| Feature | Mood Image Generator | Ad Image Generator |
|--------|----------------------|-------------------|
| Text Content | No text | Includes copy |
| Primary Use | Brand exploration, visual assets | Ad placement, conversion-oriented |
| Design Focus | Visual aesthetics, atmosphere creation | Information delivery, conversion guidance |
| Suitable Channels | Detail pages, social media | Ad platforms, e-commerce main images |
